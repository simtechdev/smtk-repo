## Quick start

Run this command on your system via root:

**CentOS 6.x**

```
yum install -y https://us.yum.smtk.st/smtk-release-6.noarch.rpm
```

**CentOS 7.x**

```
yum install -y https://us.yum.smtk.st/smtk-release-7.noarch.rpm
```
